## Installation

To install and run this tool:

```bash
  git clone https://github.com/A1a0k1a0s1h/MacTrace.git
```
Go to the file and install the requirements:

```bash
  cd MacTrace
  pip install requirements.txt
```
Once installed the requirements run using

```bash
  python MacTrace.py
```
